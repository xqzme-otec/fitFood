# FitFood — Documentation

Welcome to the maintained documentation site for **FitFood**, a "smart fridge"
web app: it computes nutrition targets (KBJU), tracks a food diary, manages a
per-user virtual fridge, ingests groceries by scanning receipts (OCR + LLM or the
FNS QR API), and recommends recipes from what the user actually has.

This site publishes the project's maintained documentation. The source lives in
the [`docs/`](https://github.com/xqzme-otec/fitFood/tree/main/docs) directory of
the [product repository](https://github.com/xqzme-otec/fitFood).
