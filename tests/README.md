# Тесты

Покрывают весь бэкенд: аутентификацию, анкету и расчёт КБЖУ, дневник питания,
холодильник, каталог и CRUD рецептов, сканирование чеков, рекомендации.

Тесты используют **изолированную SQLite-БД во временном файле** (см.
`conftest.py`) и мок-режим LLM/OCR — запущенный PostgreSQL/Docker не требуется.

## Запуск локально

```bash
pip install -r requirements.txt -r requirements-dev.txt
pytest
```

## Запуск в Docker-контейнере

```bash
docker compose exec app pip install pytest httpx
docker compose exec app python -m pytest
```

## Структура

| Файл | Что проверяет |
|------|---------------|
| `test_nutrition.py` | Расчёт BMR/TDEE/КБЖУ, пересчёт под лимит, доли приёмов (юнит) |
| `test_fridge_service.py` | Статус срока годности (юнит) |
| `test_auth.py` | Регистрация, логин, `/auth/me` |
| `test_profile.py` | Анкета, нормы, смена веса, ручная корректировка калорий/БЖУ, приёмы |
| `test_fridge.py` | Добавление (авто-категория/срок), дедуп, группировка, изменение, удаление, изоляция по пользователю |
| `test_dishes.py` | CRUD рецептов, расчёт КБЖУ на 100 г, поиск, авторизация |
| `test_diary.py` | Запись продукта/блюда, масштабирование КБЖУ, сводка дня, удаление |
| `test_receipts.py` | Фильтрация еды/не-еды, добавление подтверждённого в холодильник |
| `test_recommendations.py` | Рекомендации по холодильнику и КБЖУ |
